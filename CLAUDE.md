# kyupi-kyupi

## 1. Giới thiệu project

Đây là frontend của project liên quan đến một matching app, sử dụng các tính năng AI

## 2. Tech stack

Backend: Go

Frontend:

- NextJs, Typescript
- Shadcn ui, tailwind css v4
- Architecture: FSD
- Version control management: pnpm
- Áp dụng các best practices khi code react app: Provider pattern, Container/ presentational patter, singleton pattern, component composition, higher order components, Lazy loading...
- Các tên file dùng kebab-case

Mục tiêu về mặt frontend: cần đảm bảo tối ưu frontend, tạo trải nghiệm người dùng tốt như các trang website hàng đầu thế giới: Stripe, Unsplash, Airbnb.
